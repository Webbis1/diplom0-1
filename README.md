# CONTEXT PRIMER: Spatial Crypto Arbitrage System (Diplom Novikov)

## 1. Суть проекта
Высоконагруженный асинхронный бэкенд-сервис для пространственного криптоарбитража. Решает проблему NP-трудности поиска маршрутов с неаддитивными фиксированными комиссиями.

## 2. Математическое ядро
- **Граф:** Ориентированный. Вершины = (Биржа, Актив). Ребра = Торговые (внутри биржи) и Трансферные (между биржами).
- **Функция ценности:** Линейно-аффинная `Val(X) = a*X + b`. 
  - `a` (мультипликатор) — предельная полезность (должна быть > 1 для профита).
  - `b` (остаточная ценность) — базовая ценность минус накопленные фиксированные штрафы.
- **Алгоритм:** Локальная обратная волновая релаксация. Волна идет от целевых активов к исходным.
- **Защита от циклов:** `bitarray` (битовые маски) для хранения истории пути за O(1).
- **Топологические барьеры:** Обнуление потенциала при `a <= 1`.

## 3. Архитектура и стек
- **Язык/Парадигма:** Python 3.10+, `asyncio` (кооперативная многозадачность, Event Loop).
- **Типы данных:** `Decimal` (исключить float для финансов).
- **Транспорт:** CCXT Pro (WebSocket), паттерн Circuit Breaker, экспоненциальный бэкофф.
- **Модули:**
  1. `Connection` (Транспорт, сокеты, обработка разрывов).
  2. `Exchange` (Унификатор API через CCXT).
  3. `Analyst` (Singleton, ядро графа, полиморфная очередь `Queue[Node | Edge]`).
  4. `Manager` (Pull-on-Demand, гранулярные `asyncio.Lock` на каждую монету).
- **Бэктестинг:** Полиморфная подмена транспорта на `HistoricalApi` (чтение из БД).
- **Event Coalescing:** Дедупликация шквальных тиков через `set[Node | Edge]` (перезапись в памяти, расчет только терминального состояния).
